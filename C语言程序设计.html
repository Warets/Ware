C语言利用基础代码求矩形的周长和面积：

#include <stdio.h>

int width;         

int height;        

 

int area;          

int perimeter;     

 

int main() {

    height = 7;

    width = 5;

 

    perimeter = 2*(height + width);

    printf("该矩形的周长 = %d inches\n", perimeter);

     

    area = height * width;

    printf("该矩形的面积 = %d square inches\n", area);

 

return(0);

}

运行结果是：

该矩形的周长 = 24 inches 

该矩形的面积= 35 square inches
